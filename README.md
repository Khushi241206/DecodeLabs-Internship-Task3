## Description

The AI Recommendation System is a simple web application that provides personalized recommendations based on a user's interests. The system uses similarity matching logic to compare user-selected preferences with predefined item categories and then suggests the most relevant options. This project demonstrates the fundamental concepts behind recommendation engines used by platforms such as Netflix, YouTube, and Amazon.

## What It Does

* Takes user interests as input through an interactive interface.
* Compares the selected interests with item attributes (tags).
* Calculates a similarity score for each item.
* Ranks items based on their relevance to the user's preferences.
* Displays personalized recommendations with matching scores.

## How It Runs

1. Install the required dependencies:

   pip install -r requirements.txt

2. Start the Streamlit application:

   python -m streamlit run app.py

3. Open the local URL generated by Streamlit in your browser.

4. Select one or more interests from the available options.

5. Click the "Get Recommendations" button to view personalized recommendations.

## Conclusion

This project successfully demonstrates the core principles of recommendation systems using rule-based similarity matching. By analyzing user preferences and comparing them with item attributes, the application generates relevant recommendations in a simple and efficient manner. The project serves as a strong foundation for understanding recommendation logic and can be further enhanced using advanced machine learning techniques such as collaborative filtering, content-based filtering, and hybrid recommendation models.
